PROJECTS

Novascape

Novascape is a 2D game built in Unity that focuses around a character stuck in a galaxy that soon could get destroyed. The character starts on one planet, and travels from planet to planet trying to get the necessary resources to build a spaceship and escape the galaxy. The game features procedural planet generation for all the planets, which includes resource allocations within the planets and cave generation.

Food Database

The food database was an application built by starting with the Wikipedia dataset and extracting the food pages into our database. The user can search through all the foods in the database, and get relevant information such as where the food originates from and a link to the Wikipedia article for more information on the food. The application also features a recipe recommendation system using the user's search history and parameters that the user can set, and a map as a visualization of the origins of all the food in the database.

Grade.Me

Grade.Me was built to be a better gradebook than the ones currently offered by the school. Grade.Me includes useful features not seen in other solutions for both professors and students. Grade.Me has a simple, integrated design making it easy for a student to see all their grades in one place and an instructor to see in an easy glance how the class is doing. It also includes a grade predictor system for students, who can change future grades that have not happened yet to see how it impacts their grades. For instructors, it also includes a simple grade adjustment system, so that instructors can easily apply changes to the class and visually see how it changes the grades. Grade.Me focuses on data and visualization, and includes many graphs and statisitcs that students and instructors should expect in a grade system.
